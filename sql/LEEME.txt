===========================================
 COMO IMPORTAR LA BASE DE DATOS EN OTRO PC
 PROYECTO: Calidad el Paisa
===========================================

REQUISITOS:
- MySQL instalado (XAMPP, WAMP, o MySQL Server)
- El archivo proyecto_muebles_dump.sql

PASOS PARA IMPORTAR:

1. Abre una terminal (cmd o PowerShell) en la carpeta donde tengas MySQL

2. Si usas XAMPP:
   mysql -u root < proyecto_muebles_dump.sql

3. Si MySQL pide contraseña:
   mysql -u root -p < proyecto_muebles_dump.sql

4. Si MySQL NO está en el PATH, usa la ruta completa:
   C:\xampp\mysql\bin\mysql.exe -u root < proyecto_muebles_dump.sql

5. Verificar que se creó:
   mysql -u root -e "USE proyecto_muebles; SHOW TABLES;"

USUARIO INCLUIDO:
- Usuario: santi
- Contraseña: 1234567

Si el usuario no se crea automáticamente, ejecuta:
   CREATE USER 'santi'@'localhost' IDENTIFIED BY '1234567';
   GRANT ALL PRIVILEGES ON proyecto_muebles.* TO 'santi'@'localhost';
   FLUSH PRIVILEGES;
